Indigo

Indigo is a very deep blue with some violet tones. According to Patel , the color blue is favored by many individuals because the color represents calmness, spirituality, and may help with sleep. Many brands, like Intel and Samsung main colors are blue. The color Indigo is a more cold and neutral color than a warm color. Text should never be blue, unless the text is also a link. According to The Psychology of Colors, the color Indigo promotes concentration and meditation. The color Indigo would mainly be used for websites that have content about sleep, meditation, or spirituality. According to RapidTables, the Hex code for Indigo is #4B0082 and the RGB code is (75, 0, 130).

Indigo is a darker blue, while Turquoise is a lighter blue.
