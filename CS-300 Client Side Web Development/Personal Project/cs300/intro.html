XHTML Rules

  • Elements must nest symmetrically.
  • Element names are case sensitive.
  • End tags are required.
  • Empty elements are signified with a closing slash.
  • Attribute values must be contained in quotation marks.
  • Attribute value pairs must be written in full.

Definitions

World Wide Web Consortium (W3C)
"The W3C sets standards for markup languages and provides an open, nonproprietary forum for industry and academic representatives to add to the evolution of HTML" (Sklar, 41).
Semantic Markup
"Descriptive markup that identifies the intended use of document sections" (Sklar, 41).
Element
"A pair of HTML tags containing content" (Sklar, 40).
Attribute
"Code added to an element to provide more information about the element" (Sklar, 40).
Metadata
"Information about the document itself, such as how to present the document, or what other documents are related to the current one, such as style sheet" (Sklar, 41).
