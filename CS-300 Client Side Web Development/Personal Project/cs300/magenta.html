Magenta

Magenta is a mixture of purple and red, with pink tones. According to Cyriac, red catches the human eye and purple increases creativity. “Buy Now” or “Book Now” would be in red because the color would catch the human eye faster. The color Magenta would be more of a cooler tone than a warmer tone. According to The Psychology of Colors, the color Magenta encourages compassion and intuitive thinking. In addition, the color Magenta is used for websites that contain spring content, like flowers and gardening. According to RapidTables, the Hex code for Magenta is #FF00FF and the RGB code is (255,0,255).
